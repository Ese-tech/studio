# **App Name**: CareerWaveApp

## Core Features:

- Job Listing: Display job postings with filtering and search capabilities (title, location, category, contract type).
- User Authentication: Allow users to create accounts and manage their profiles with Firebase Authentication.
- Profile Management: Enable candidates to create profiles and upload their CVs using Firebase Storage.
- Job Application: Allow users to apply for jobs and track their application status.
- Employer Dashboard: Provide employers with a dashboard to create, edit, and delete job postings.
- Application Management: Allow employers to view applications and candidate profiles.
- Content Moderation Tool: AI-powered tool to detect and flag inappropriate content in job postings and user profiles, assisting administrators in content moderation.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for trust and professionalism.
- Background color: Light grayish-blue (#E6F3FF).
- Accent color: Sky blue (#87CEEB) for interactive elements and highlights.
- Body font: 'Inter' (sans-serif) for clean, modern readability.
- Headline font: 'Space Grotesk' (sans-serif) for a contemporary, slightly techy feel.
- Use modern, clear icons to represent job categories and actions.
- Subtle animations for transitions and loading states to improve user experience.
